# ☁️ Cloud Job Applicant Management System

## 📖 Project Overview

The Cloud Job Applicant Management System is a full-stack web application developed as part of my AWS Cloud & DevOps Internship.

This application allows applicants to register, upload resumes, apply for jobs, and manage their profiles. Administrators can manage applicants, job postings, and applications through an admin dashboard.

The project demonstrates practical implementation of Full Stack Development, Cloud Computing, Docker, CI/CD, and AWS Services.

---

# 🎯 Objectives

- Develop a complete job application management system.
- Learn Full Stack Web Development.
- Deploy applications using Docker.
- Use AWS cloud services for hosting and database.

# ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/tarunprojects1-maker/Cloud-Job-Applicant-Management-System.git
```

Backend:

```bash
cd backend
npm install
npm start
```

Frontend:

```bash
cd frontend
npm install
npm run dev
```

---

# 🔐 Environment Variables

Create a `.env` file in the backend folder.

Example:

```env
PORT=5000

DB_HOST=YOUR_DATABASE_HOST
DB_PORT=3306
DB_USER=YOUR_DATABASE_USERNAME
DB_PASSWORD=YOUR_DATABASE_PASSWORD
DB_NAME=YOUR_DATABASE_NAME

JWT_SECRET=YOUR_SECRET_KEY
```

**Never upload your real `.env` file to GitHub.**
